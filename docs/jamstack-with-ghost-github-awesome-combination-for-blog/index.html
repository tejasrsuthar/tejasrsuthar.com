JamStack with Ghost & GitHub: Awesome combination for a blog

headless cms • Sep 01, 2019

I was really in a need of tech stack where I can write technical blog but it was really time consuming to run a Wordpress blog fore multiple reasons like a it was really heavy for normal tech blogger like me. As I don't want extra clutter to maintain a CMS site. I was always wanted a simple blogging system that is flexible enough that I don't have to spend too much time to maintain it.

Initially I was using a static site generator like Jekyll. It was really good site generator from markdown files. I worked on it for few months and developed a site as well from it and host it to github. After few time I was busy working on other freelancing projects and really didn't look at it.

After couple of years, when I look at the static site generator era, its completely different storm here. Everyone is talking about JamStack and static site generator and there also Ghost CMS was available. It's the best headless CMS I've ever worked with. Typing on its editor is just a breeze.
